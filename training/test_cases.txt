1. При попытке заказать автомобить в недоступной локации :
	- В форму вводим Grodno, нажимаем поиск
	- Результат: Please search for a location and select from the available options.

2. При вводе возроста водителя:
	- В поле для возроста вводим 16 и нажимаем "Find a car"
	- Результат: Please enter the main driver's age above

3. Если при заказе автомобиля указывать время которое уже прошло:
	- Например выбираем время - 10:00, когда сейчас 15:30 и нажимаем "Find a car"
	- Результат: Sorry, it is too late to make this booking online. Currently we require 2 hours notice to make a booking.	
	
4. При попытке взять автомобиль на слишком длительное время:
	- В поле для выбора дат бронирования автомобиля и возврата. Вводим 10:00, в 15:40
	- Результат: There are no vehicles available for a rental duration longer than 60 days.

5. При аренде автомобиля можно выбрать дополнительные города в которые можно поехать, за каждый город цена увеличивается на €10. Если водителей будет больше одного цена возростает на €127.5 за каждого.

6. Перед оплатой ессть надпись о том, что с карты автоматически будут счислены €1000, как возврощаемый долг.

7. На данном сайте нет возможности вернуть автомобиль в другой стране (отличной от той где автомобиль забронирован)

8. При изменении валюты стоимость в представленных возможных вариантах аренды отображается в выбранной валюте
	- Заполняем форму и нажимаем "Find a car"
	- Меняем валюту с € на £
	- Результат: на странице с вариантами для аренды стоимость будет отображаться в £

9. На 3ем этапе бронирования автомобиля необходимо указать пользовались ли этим сайтом до этого, а также ввести Email иначе отображется ошибка
	- Ничего не заполняя нажимаем на Continue
	- Please select an option. Please fill out this field.

10. Если при оплате ввести не действительные данные, выведет ошибку
	- Card Number - 0000000000000000
	- Expiry Date - mm:yyyy - 10:2000
	- Name on Card - asdf
	- Security Code - 456
	- Нажимаем Continue
	- Результат: 	Please enter a valid card number.
			Please enter a valid expiry date.
